Обо мне

Я Егор Виссарионов, Python разработчик, создающий кастомные боты для автоматизации сложных задач. Мои проекты — это уникальные решения, написанные с нуля на Python, без шаблонов и конструкторов. Я превращаю идеи в эффективные инструменты.

Мой опыт включает веб-скрейпинг, разработку Telegram-ботов, интеграцию API и управление данными. Я фокусируюсь на производительности, масштабируемости и удобстве для пользователей.

Ключевые навыки:
  • Python (Selenium, BeautifulSoup, Requests, Telethon, aiogram)
  • Веб-скрейпинг и парсинг данных
  • Telegram-боты с ролями и интеграциями
  • Автоматизация бизнес-процессов
  • PostgreSQL, Redis, FastAPI, Flask

Портфолио

Checkers Analyzer Bot

Telegram-бот для анализа игр в шашки, использующий движок GNUBG. Хранит состояние в Redis, данные в PostgreSQL, построен на aiogram. Поддерживает роли, интеграцию с ЮKassa, промокоды и генерацию Excel-отчетов с расчетом среднего балла по играм. GitHub

Car Auction Bot

Бот-посредник для аукционов автомобилей в Telegram. Публикует лоты в группе с деталями и ссылками, предоставленными админами, и позволяет пользователям делать ставки. Построен на Python с использованием aiogram. GitHub

Uchet Bot

Многофункциональный Telegram-бот для учета инструментов, материалов и сотрудников на стройке. Включает FastAPI, aiogram, Flask-Admin и PostgreSQL. Поддерживает роли, многоязычность (RU, UZ, TG), документооборот, отчеты и интеграцию с Telegram-группами. GitHub

iPhone Inventory Bot

Telegram-бот с интегрированным веб-приложением для учета iPhone по штрих-кодам. Работает на вебхуках, включает встроенный сканер штрих-кодов. Построен на Python с aiogram для эффективного управления инвентарем. GitHub
